FLOW & USER EXPERIENCE

SEAMLESS USER JOURNEY:

New User:
  1. Opens app → Splash screen
  2. "Get Started" → Onboarding (3 slides)
  3. "Create Account" → Auth with Clerk
  4. Account created → Dashboard (empty state)
  5. "Explore Courses" → Course Catalog
  6. Selects course → Course Detail
  7. "Enroll" → Enrolled, shows modules
  8. Taps module → Module Detail, shows lessons
  9. Taps lesson → Lesson (Learn + Try It)
  10. "Start Quiz" → Quiz (5 questions)
  11. Completes quiz → Results (celebration)
  12. "Continue" → Back to Dashboard
  13. Profile → View achievements

Returning User:
  1. Opens app → Splash (auto redirect)
  2. Dashboard (shows enrolled courses)
  3. "Continue Learning" → Resume lesson
  4. Or "Your Courses" → Select course
  5. Same learning flow (lesson → quiz → results)

Admin User:
  1. Dashboard (with admin access)
  2. Admin Dashboard → Manage courses
  3. Create/Edit/Delete courses, modules, lessons
  4. View user statistics
  5. All with proper access control

Invalid Route:
  → 404 page with "Darkness vs Light" theme
  → CTAs: "Go Home" or "Explore Courses"
